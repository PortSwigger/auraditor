Auraditor is a Burp Suite extension developed for security testing Salesforce Lightning and Aura framework applications. This tool has comprehensive audit capabilities along with advanced action management, and context editing. Always use this tool responsibly and ensure you have proper authorization before testing any systems. The tool is provided without any support, warranty, or liability from the developer.

Features

Request Editor:
Here are the main features of the Request Editor:

  • View and edit Aura actions in HTTP requests
  • Add and remove actions using tabs
  • Edit controller names and method names
  • Modify JSON parameters for each action
  • Handle invalid JSON according to user preference
  • Copy, cut, and paste in text fields
  • Toggle line wrapping for better readability

Base Requests Management:
The Base Requests Management feature allows you to:

  • Save multiple base requests from HTTP history
  • Tag requests with custom names
  • Use saved requests for security testing operations

Discovery Operations:
The Discovery Operations feature can:

  • Find Aura controllers and methods from JavaScript files
  • Discover Lightning Web Component (LWC) endpoints
  • Extract API routes from application files
  • Search for objects by name in the application

Route Testing:
With Route Testing, you can:

  • Test discovered routes automatically
  • Categorize routes by response type
  • Export results to files

Salesforce ID Tools:
Salesforce ID Tools lets you:

  • Analyze Salesforce ID structure and format
  • Convert between 15-character and 18-character IDs
  • Generate sequential Salesforce IDs
  • Create custom ID payload generators for Burp Intruder
  • Change decimal values in Salesforce IDs

Usage

Auraditor has a simple and intuitive interface that makes using these features easy, even for beginners. The Request Editor allows for the viewing and editing of Aura actions in HTTP requests. Actions can be added or removed using tabs. Controller names, method names, and JSON parameters for each action can be modified. The user can choose how to handle invalid JSON. Useful text editing features are available like copy, cut, and paste. Line wrapping can be toggled for better readability.

In terms of Base Requests Management, multiple base requests can be saved from HTTP history, tagged with custom names, and then be used for security testing operations.

For Discovery Operations, the extension can find Aura controllers and methods from JavaScript files, discover Lightning Web Component (LWC) endpoints, extract API routes from application files, and can even search for specific objects by name within the application.

The Route Testing feature can test discovered routes automatically, categorize them by response type, and the results can then be exported to files.

Finally, the Salesforce ID Tools are helpful for analyzing the Salesforce ID structure and format, converting between 15-character and 18-character IDs, generating sequential Salesforce IDs, creating custom ID payload generators for Burp Intruder, and changing decimal values in Salesforce IDs.
